# Hide-n-Seek-Pale-Events-
A simple Minecraft Hide &amp; Seek plugin inspired by Hypixel-style gameplay. It features basic game flow, block selection, scoreboards, and GUI support. The code is intentionally lightweight and easy to extend, making it a solid foundation, though it currently lacks advanced systems and broader versatility found in larger production plugins.
